## Prerequisites

* MySQL/MariaDB Server
* Python 3.9+
* Python Modules
    * `PySide2` - GUI Library
    * `qtawesome` - Icon Library
    * `qt-material` - Material Design Stylesheet for PySide2
    * `mysql-connector-python` - SQL Library to connect to MySQL Server

## How to run ?

Execute `python main.py`
